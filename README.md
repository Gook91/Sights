Приложение для показа достопримечательностей на карте и фото, сделанных пользователем. Проект работает с системой разрешений Android, запрашивая опасные разрешения у пользователя. В качестве источника карт была задействована библиотека Osmdroid, данные о достопримечательностях получены с сервера OpenTripMap, фотографии, сделанные пользователем, сохраняются на устройстве после получения разрешения на доступ к памяти.
